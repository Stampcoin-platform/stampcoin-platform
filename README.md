# To‑Do List (LocalStorage)

A small, vanilla JavaScript to‑do list application that persists data in the browser's `localStorage`.

Features:
- Add new todos
- Edit existing todos (double-click or press Enter)
- Toggle complete / active
- Delete todos
- Clear completed
- Filter: All / Active / Completed
- Persistent storage using `localStorage` (key: `todos-v1`)
- Responsive and accessible (basic ARIA)

How to run:
1. Download the files (index.html, style.css, app.js).
2. Open `index.html` in your browser (no server required).
3. The app saves your todos in the browser; clearing browser storage will remove them.

Optional improvements:
- Add timestamps and sort options.
- Add drag-and-drop reordering.
- Sync with a backend or use cloud storage.
- Add animations and improved styling.
